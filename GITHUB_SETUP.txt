GITHUB SETUP - Step by step
===========================

PART 1 - On github.com (in your browser)
----------------------------------------
1. Log in to https://github.com
2. Click the + button (top right) -> New repository
3. Repository name: wepl-report-viewer  (or any name you like)
4. Choose PRIVATE (recommended)
5. Do NOT check "Add a README" (you already have files)
6. Click Create repository
7. Copy the repository URL shown (looks like):
   https://github.com/YOUR_USERNAME/wepl-report-viewer.git


PART 2 - In PowerShell (in this project folder)
-----------------------------------------------
Open a NEW PowerShell window (so Git is recognized), then run:

cd "c:\Users\shayn\OneDrive\Documents\Cursor Projects\ISFPT\New folder"

git init
git add .
git status
git commit -m "Initial WEPL report viewer"
git branch -M main
git remote add origin PASTE_YOUR_REPO_URL_HERE
git push -u origin main

Replace PASTE_YOUR_REPO_URL_HERE with the URL from Part 1.


PART 3 - First-time login
-------------------------
GitHub may ask you to sign in in the browser.
Follow the prompts.


PART 4 - Tell your colleague
----------------------------
Send them the repo link (they need access if private).
They click Code -> Download ZIP, or clone the repo.
Then: edit config.txt, double-click START_VIEWER.bat.


TROUBLESHOOTING
---------------
- "git is not recognized" -> Close PowerShell and open a NEW window
- "remote origin already exists" -> run: git remote remove origin
  then add origin again with the correct URL
